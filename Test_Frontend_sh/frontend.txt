HTML(Hyper Text Markup Language)
  markup을 이용해서 해당 컨텐츠의 용도를 알려줌(브라우저)
  렌더링 엔진이 마크업기능을 해석해서 출력

  대소문자를 가리지 않음 --> 소문자를 사용

     <태그명 속성명="속성값">... </태그명> :
     <태그명>

  다른 태그를 포함하여 사용되거나 포함되어 사용된다.
  모든 태그들은 자기만의 고유스타일이 있어서 CSS를 이용해서
  바꿀 수 있다.

  블록 요소(block tag) ==> 포함하여 사용, 패딩, 마진을 가질 수 있음
     hn, p, div, blockquote, div, pre ...
    hr: 가로선
    태그의 폭이 컨테이너의 100%
       (다른 요소와 영역을 나누지 않음)


  인라인 요소(inline tag)==> 포함되어 사용,  패딩, 마진을 가질 수 없음
    태그의 폭이 컨텐츠의 폭이다.
    a : 다른 블록요소에 포함되어 사용된다.
    폼태그:
    text : 인라인 요소의 한종류이다.

    br:줄바꿈, 오픈태그
