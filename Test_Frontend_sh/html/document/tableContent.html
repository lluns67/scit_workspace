테이블을 구성하는 요소

  • table : 테이블을 만드는 Root요소(행, 열, 셀)
  • thead, tbody, tfoot: 테이블의 큰 카테고리 만드는 요소,
    생략하면 tbody는 브라우저의 렌더링 엔진이 자동생성
  • tr: 행표현
  • th: 제목열(가운데정렬, 굵게), td: 일반열
