CSS 선택자(Selector)

HTML 문서에 배경색이나 글꼴 등의 서식을 적용하기 위해서는 적용 대상을 선택해야 한다.
선택자는 문서의 특정 요소를 다양한 방식으로 선택하도록 도와준다.
선택자는 우선순위(특이값)가 존재
css를 사용하기 위해서는 여러 방식이 있다. 

    😊😊특잇값 순서
        1) important
        2) inline style : 1000
        3) id : 100 (현재 문서 내에서 동일한 아이디는 가질 수 없다.) => js
        4) class : 10 (현재 문서 내에서 동일한 클래스는 여러개 가질 수 있음. 하나의 태그에 여러 클래스도 가능.)
        5) tag : 1
        6) *: (universal) 우선순위 없음. 보통 최상단에 두어서 태그가 가지는 기본값을 없애는 용도. 사용을 권잘하지 않음.

    😊😊선택자 종류
        1) 아이디 선택자
        2) 클래스 선택자
        3) 태그 선택자 
        4) * 유니버설 선택자
        5) 계층선택자 : 자식(>), 후손(띄어쓰기), 형제(+)
        6) 속성 선택자 : input[type ~= ]

  • inline 스타일 : 태그에 직접 스타일을 적용
