css 선택자

html 문서에 배경색이나 글꼴등의 서식을적용하기 위해서는 적용 대상을 선택해야한다 선택자는 문서의 특정 요소를 다양한 방식으로 선택하도록 도와준다.
선택자는 우선순위(특잇값)가 존재
css를 사용하기위해서는 여러 방식이 있다. 

      1) 아이디 선택자
      2) 클래스 선택자
      3) 태그 선택자
      4) * 유니버셜 선택자
      5) 계층선택자 : 자식(>), 후손(띄어쓰기), 형제(+)
      6) 속성 선택자 : input[type="text"]

  • 외부파일 참조 (*.css)
  • inline: 태그에직접 스타일을 적용
