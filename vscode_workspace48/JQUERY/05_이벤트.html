[J-Query 이벤트]

  1. 이벤트 관련 함수
    • on() : 이벤트 연결
    • off() : 이벤트 해제
    • click() : 클릭 이벤트 연결
    • dblclick() : 더블 클릭 이벤트 연결
    • mouseenter() : 마우스가 요소에 진입할 때 이벤트 연결
    • mouseleave() : 마우스가 요소에서 벗어날 때 이벤트 연결
    • hover() : 마우스가 요소에 진입/벗어날 때 이벤트 연결
    • focus() : 포커스가 요소에 진입할 때 이벤트 연결
    • blur() : 포커스가 요소에서 벗어날 때 이벤트 연결
    • submit() : 폼이 전송될 때 이벤트 연결
    • keydown() : 키보드 키를 누를 때 이벤트 연결
    • keyup() : 키보드 키에서 손을 뗄 때 이벤트 연결
